# Engineering Intelligence — companion code

Runnable companions to the book _Engineering Intelligence_. Three
projects live here.

## `notebooks/`

Twenty analysis notebooks that reproduce the book's worked examples and
demonstrate reusable analytical patterns. Each is a `.py` file with
`# %%` cell markers — open one in Jupyter, VS Code, PyCharm, or run it
directly as a script. Prefer traditional `.ipynb`? They're pre-built
under `notebooks/ipynb/`, regenerated by `scripts/build_ipynb.py`.

## `utils/`  (`engintel_utils` package)

Reusable analytical utilities used across the notebooks:

- `stats.py` — bootstrap CIs, two-sample z, power / sample-size, O'Brien-Fleming boundary, Beta-Binomial posterior.
- `reconstruction.py` — toy Engineering-Change reconstruction from
  event streams with confidence bands.
- `metrics.py` — metric specs, semantic versioning, composite metric
  scaffolding, z-scoring, version comparison.
- `graph.py` — small DAG implementation with topological order,
  ancestors, descendants, critical path, parallelism ratio.
- `blast.py` — graph-derived reach / exposure, reversibility, composite
  score.
- `monte_carlo.py` — component-wise cost distributions and Monte Carlo
  aggregation.
- `governance.py` — use-classification and routing/cadence lookups
  matching Appendix D.

## `synth/`  (`engintel_synth` package)

Deterministic synthetic-telemetry generator: pull-request events,
reviews, CI runs, deployments, incidents, teams, actors (human / bot /
agent), and reconstructed Engineering Changes with confidence bands.

Usage:

```
pip install -e companion/synth
python -m engintel_synth --days 30 --teams 40 --seed 42 --out data/
```

Same seed produces byte-identical output. See `synth/SCHEMA.md`.

## Install everything

```
pip install -r companion/requirements.txt
pip install -e companion/utils
pip install -e companion/synth
python3 companion/scripts/build_ipynb.py   # regenerate .ipynb from .py
```

`numpy`, `scipy`, and `matplotlib` are the only external dependencies;
the utils and synth packages otherwise use the standard library.
